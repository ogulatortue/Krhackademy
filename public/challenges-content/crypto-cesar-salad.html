Contexte

Vous êtes analyste en cybersécurité pour la société "Innovatech Dynamics". Suite à une alerte de sécurité, vous auditez les communications internes. Votre attention est attirée par un message suspect échangé entre deux employés sur une messagerie d'entreprise, dans un canal supposé anodin. Le message est court, semble codé, et a été posté juste avant une fuite de données mineure.

La direction craint que ce message ne soit un test avant une opération de plus grande envergure ou la preuve d'une menace interne. Votre mission est de casser ce code simple pour comprendre la nature de cette communication et identifier le risque potentiel.
