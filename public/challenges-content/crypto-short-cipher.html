Dossier de Terrain

Une cellule de renseignement a déployé un dispositif de signature rapide destiné à accélérer le débit des messages dans un canal secret. Pour économiser les cycles CPU, le développeur a choisi un exposant privé $d$ très petit. Le dispositif a rempli sa fonction — jusqu'à ce que les analystes découvrent que les petits $d$ peuvent être divulgués via les fractions continues.

Briefing de Mission

Votre mission est de casser la clé publique afin de déchiffrer le message chiffré.

  • Analyse : La sécurité d'une clé RSA peut être compromise si l'exposant privé d est particulièrement petit. Cette vulnérabilité peut être exploitée à l'aide de l'algorithme des fractions continues.
  • Objectif : Récupérer d, puis déchiffrer le contenu de cipher.txt pour obtenir le drapeau.

Indice : Challenge de Wiener : ce RSA a été généré avec un exposant privé d très petit.
