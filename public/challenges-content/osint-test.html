Contexte

Une développeuse anonyme, connue uniquement sous le pseudonyme de "Kaelis", a récemment mis en lumière une faille de sécurité majeure dans un logiciel gouvernemental. Après avoir publié ses découvertes de manière responsable, elle a effacé toute trace de son passage, ne laissant derrière elle qu'un unique message énigmatique sur un forum spécialisé avant de disparaître.

Votre mission, en tant qu'enquêteur OSINT, est de remonter sa piste numérique pour découvrir le nom de code du projet personnel sur lequel elle travaillait avant de s'évanouir dans la nature. Ce projet est suspecté d'être la véritable raison de sa disparition.

Unique Artefact : Le Dernier Message

Message de "Kaelis" (transcription)

"Mon dernier commit public sur la plateforme du félin octogonal scelle mon départ. Cherchez mon alias, 'kaelis_7', là où le code se forge. La photo prise depuis mon balcon de vigie, face au Lion de Belfort, était mon seul indice visuel. Le projet 'Opale' n'était que la partie visible de l'iceberg. Si vous êtes assez curieux, vous trouverez le vrai nom du projet qui m'a consumé."

Objectif et Procédure

Votre objectif est de découvrir le nom de code du projet secret de Kaelis. Contrairement aux challenges précédents, toutes les informations ne vous sont pas fournies. Vous devrez activement chercher les indices sur internet.

  • Décrypter le point de départ : Analysez le message pour identifier la plateforme web, le pseudonyme à chercher et le lieu géographique clé.
  • Pister l'alias : Retrouvez le profil de "Kaelis" sur la plateforme identifiée. Une analyse minutieuse de son profil, de ses projets (repositories) et de son activité est nécessaire. Le message mentionne un "dernier commit public" comme étant la clé.
  • Analyse d'Artefact Numérique : Le dernier commit pourrait contenir un indice caché. Il est possible que des informations aient été intentionnellement laissées (ou retirées) dans les fichiers. Pensez à l'analyse de métadonnées (EXIF pour les images, par exemple) et à l'historique des modifications.
  • Pivoter vers une nouvelle source : L'indice que vous trouverez vous mènera probablement vers une autre plateforme, plus professionnelle, où son identité réelle pourrait être révélée.
  • Synthétiser les informations : Une fois sur son profil professionnel, explorez-le en détail en gardant en tête les indices précédents pour localiser la section contenant le nom de code du projet.

Flag : Pour valider ce challenge, le flag à soumettre est le nom de code exact du projet que vous avez découvert.

Solution Détaillée

Cliquez pour révéler la solution
  1. Étape 1 : Décryptage du message initial
    • "La plateforme du félin octogonal" est une référence directe à GitHub et son logo, l'Octocat.
    • L'alias à chercher est kaelis_7.
    • Le lieu clé est Belfort, identifié par la mention de la célèbre statue du "Lion de Belfort".
  2. Étape 2 : Enquête sur GitHub
    • En cherchant kaelis_7 sur GitHub, on trouve un profil. Le nom affiché sur le profil est Elara Vasseur. La photo de profil correspond à la description : une vue du Lion de Belfort depuis un appartement en hauteur.
    • Parmi ses dépôts (repositories), l'un d'eux, nommé dotfiles, a un commit très récent. Le message du dernier commit est : "Nettoyage de printemps. Suppression des métadonnées avant l'archivage."
  3. Étape 3 : Analyse de l'historique du commit et des métadonnées
    • Ce message de commit est un indice majeur. Il suggère que la version précédente d'un fichier contenait des métadonnées importantes.
    • En regardant l'historique de ce commit, on voit que le fichier modifié est l'avatar de l'utilisateur (par exemple, avatar.jpg).
    • Il faut donc télécharger la version du fichier avant ce dernier commit.
    • En utilisant un outil d'analyse de métadonnées EXIF, on inspecte cette ancienne version de l'image. Dans le champ "Commentaires", on trouve une nouvelle piste : Coord: 47.6391° N, 6.8636° E. Le vrai projet est dans le pro. Linkedin, section bénévolat.
  4. Étape 4 : Pivot vers LinkedIn
    • L'indice nous dirige vers LinkedIn et nous donne son vrai nom, trouvé sur GitHub : Elara Vasseur.
    • Une recherche sur LinkedIn pour "Elara Vasseur", potentiellement en ajoutant "Belfort" ou "Développeuse", permet de trouver son profil professionnel.
  5. Étape 5 : Découverte du Flag
    • L'indice EXIF précisait de regarder la section "Bénévolat" (Volunteering) de son profil.
    • Dans cette section, on trouve une entrée décrivant sa contribution à un projet de cyber-défense citoyenne. Le nom de code est mentionné ici.
