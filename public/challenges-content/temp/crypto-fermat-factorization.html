Dossier de Terrain

Un laboratoire de terrain dans la région frontalière a mal configuré son générateur de nombres aléatoires matériel (Hardware RNG) lors du démarrage d'un ancien terminal sécurisé. Deux nombres premiers ont été générés de manière anormalement proche — une négligence de fabrication, ou peut-être une tentative par un agent de bas niveau de laisser un exploit pour des alliés "de confiance".

La paire de clés résultante a été archivée puis a fuité dans notre dépôt d'entraînement. Votre analyse est requise.

Clé publique (pub_pub.txt)

Briefing de Mission

Votre mission est de casser la clé publique afin de déchiffrer le message chiffré.

  • Analyse : La sécurité d'une clé RSA repose sur la difficulté de factoriser son module $n$. Toute faiblesse dans la génération des nombres premiers $p$ et $q$ peut être exploitée.
  • Objectif : Déchiffrer le contenu de cipher.txt pour récupérer le flag.

Indice : Challenge de factorisation de Fermat : $p$ et $q$ sont anormalement proches.

Flag

Pour valider ce challenge, le flag à soumettre est le message déchiffré en clair.
