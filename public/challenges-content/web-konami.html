Dossier de Terrain

Nos agents ont découvert un serveur dormant appartenant à un hacker de la vieille école, une légende des années 90 connue sous le pseudonyme de "Master K". Ce dernier était réputé pour son amour des jeux vidéo classiques, une passion qu'il intégrait souvent dans ses systèmes de sécurité.

Une seule page web est active sur le serveur. Elle semble être une sorte de coffre-fort numérique. Les rumeurs disent que "Master K" a protégé l'accès à ses archives secrètes avec un code d'entrée inspiré de son passe-temps favori. Le drapeau est la clé de ces archives.
