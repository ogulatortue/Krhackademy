Dossier de Terrain

Un administrateur système excentrique a décidé de "sécuriser" l'accès à un fichier sensible de manière peu orthodoxe. Au lieu d'un mot de passe, il a créé une interface interactive qu'il juge inviolable.

La page présente un défi : 200 boutons, dont un seul est le bon. Un clic sur le mauvais bouton et le jeu se réinitialise complètement, changeant la position du bouton correct. L'administrateur est convaincu que son système, basé sur la pure chance, est infaillible...

Briefing de Mission

Votre mission est de trouver un moyen de cliquer sur le bon bouton du premier coup pour récupérer le drapeau.

Pour valider ce challenge, le flag à soumettre est celui qui apparaît dans la console du navigateur après avoir cliqué sur le bon bouton.
